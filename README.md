### 프로그램 개요
퇴근 후 직장인에게 편의를 제공하기 위한 웹 사이트


### 주요 기능
- 회식 식당/미용실/네일샵 검색 및 상세 정보 제공 기능
- 유튜브 사용자 선택에 따른 음악 동영상 추천 기능
- 개인 찜 리스트 기능
- 사용자의 입력에 따른 경로 찾기 및 주변 편의 시설 검색 기능
- YouTube API(동영상 추천), Google Direction API(경로 찾기) , Kakao Maps (편의 시설 검색)
